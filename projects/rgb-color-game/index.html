About The Game

The RGB Color Game is a simple guessing game where you're provided with an RGB color value and four different color swatches. Your goal is to select the color swatch that matches that RGB value you've been given. Answer correctly and your score and current streak will go up. Answer incorrectly and your score will go down and your current streak will be reset to zero.

RGB color values are written as rgb(red, green, blue), with each color parameter being replaced with a value between 1 and 255 that represents the intensity of that color in the final result. Use the sliders below to check it out in action.

Knowing this, we can begin to deconstruct the RGB color values we're given in the game to help improve our answers. Head on over to the game and give it a shot!
